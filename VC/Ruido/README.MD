# Atividade - Ruído:

Este projeto tem como objetivo desenvolver um programa para a aplicação de diferentes tipos de ruídos, suavizações e detectores de borda. Explorando os parâmetros de cada técnica de ruídos.

### Instalando dependências:

Execute os seguintes comandos para instalação das bibliotecas nescessárias

WINDOWS:

Utilize python 3.10 ou 3.11

```
pip install opencv-python numpy scikit-image PyQt5
```

LINUX:

Caso apareça erro Qt utilize:

```
sudo apt install python3-pyqt5
```

```
sudo apt install qtbase5-dev
```

Ou caso esteja utilizando uma distro mais recente tente:

```
pip install opencv-python numpy scikit-image PyQt6
```

## Executando o programa:

Execute o comando abaixo:

```
python .\main.py
```
